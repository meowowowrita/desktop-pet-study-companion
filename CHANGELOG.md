# 版本记录

## v1.0.0 (2026-06-27) — MVP 骨架

### ✅ 已完成

- Electron + React + TypeScript + Vite 项目骨架
- 透明无边框桌面宠物窗口
- 宠物视频播放（asset:// 自定义协议，支持 Range 请求）
- JS 拖动窗口（左键按住拖动）
- 右键弹出快捷菜单（喂食/洗澡/清理/玩耍/学习/控制面板）
- 控制面板（状态/专注/商店/背包/设置 标签页）
- 宠物状态系统（7维：饥饿/清洁/心情/精力/健康/无聊/亲密度）
- 时间衰减 + 离线追赶
- 专注计时器（开始/暂停/继续/放弃 + 金币结算）
- 宠物互动（喂食/洗澡/清理/玩耍 — 纯函数核心）
- 本地存档（JSON + 3份自动备份 + 版本迁移接口）
- 首次启动自动送小猫
- 中英双语架构预留
- 14种宠物数据配置（普通→传说）
- 5种性格系统配置
- Windows 开发环境可运行
