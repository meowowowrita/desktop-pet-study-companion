# 素材管理说明

## 文件夹结构

```
assets/
  pets/          ← 所有宠物视频和图片
    [宠物名]/     ← 例如：cat, dog, frog, bunny...
      baby/      ← 幼年阶段的视频
      child/     ← 少年阶段的视频（后期启用）
      adult/     ← 成年阶段的视频（后期启用）
      portrait.png ← 商店/状态面板用的静态头像（可选）
  foods/         ← 食物图标（PNG）
  toys/          ← 玩具图标（PNG）
  outfits/       ← 装扮图片（PNG，后期启用）
  backgrounds/   ← 桌面背景（PNG，后期启用）
  ui/            ← 界面图标（PNG）
```

## 宠物视频命名规则

每个状态对应一个视频文件，名字固定，放到对应成长阶段的文件夹里：

| 文件名 | 对应状态 | 什么时候播放 |
|--------|----------|------------|
| `idle.mp4` | 待机/平静 | 默认状态，什么都不做时循环播放 |
| `happy.mp4` | 开心 | 刚喂食完、刚洗完澡、刚完成学习 |
| `sad.mp4` | 悲伤/疲惫 | 饥饿、脏、心情很低时 |
| `hungry.mp4` | 饿了 | 饥饿度低于 30 时 |
| `dirty.mp4` | 脏了 | 清洁度低于 30 时 |
| `sleeping.mp4` | 睡觉 | 精力很低或用户长时间不互动 |
| `playing.mp4` | 玩耍 | 用户点击"玩耍"后播放 |
| `angry.mp4` | 闹脾气 | 长时间被忽略、心情极低 |

## 视频格式要求

- 格式：`.mp4` 或 `.webm`
- 如果要透明背景：用 `.webm`（VP9 编码，带 Alpha 通道）
- 如果不需要透明背景：普通 `.mp4` 即可
- 建议分辨率：300×300 到 500×500 像素
- 建议时长：2-5 秒的循环动画
- 视频会自动循环播放

## 如何新增一只宠物

假设你要加一只小青蛙，按以下步骤：

1. 在 `pets/frog/` 下创建 `baby`、`child`、`adult` 三个文件夹
2. 至少准备 `baby/idle.mp4`（幼年待机动画，这是必须的）
3. 准备其他状态的视频（可选，缺失会自动用 idle 代替）
4. 可选：放一张 `portrait.png` 作为商店头像
